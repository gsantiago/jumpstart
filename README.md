## Usage

Clone the repository

```git clone https://github.com/gsantiago/jumpstart.git```

Install all dependencies

```npm install```

If you don't have Gulp globally installed, then install it:

```npm install -g gulp```

Now, it's just use the available gulp commands:

```gulp watch```

```gulp lint```

```gulp sprite```

```gulp css```

```gulp js```
